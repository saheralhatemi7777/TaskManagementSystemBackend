# Task Management System Backend

A full-featured Task Management System backend developed using .NET Core, implementing Clean Architecture and modern design patterns.

## Technologies & Tools
- C#, ASP.NET Core Web API
- Clean Architecture, CQRS (MediatR), Repository & Unit of Work
- Entity Framework Core, SQL Server
- AutoMapper
- JWT Authentication
- Swagger, Postman
- Git & GitHub

## Features
- Full CRUD operations for tasks
- User Management with role-based access
- JWT authentication
- CQRS with MediatR
- API documentation with Swagger
